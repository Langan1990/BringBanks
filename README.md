## Introduction

This project provides the design and documentation for the datasets "Galway City Recycling Bring Bank Locations"
which is available at [data.gov.ie]https://data.gov.ie/dataset/galway-city-recycling-bring-bank-locations

## About the data

This dataset was received in Comma Separated Values (CSV) format, and was downloaded from [*insert page name*](insert url).
The CSV file contains 13 rows, the first being a header row with the names of each field.
There are six values on each line, which are as follows:
    - **year**: the year that the car was purchased.
    - **price**: the price of the car.
